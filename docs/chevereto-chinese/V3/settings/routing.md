# 路径

`/dashboard/settings/routing`

## 图像路径

确定图像路径。

|类型 |例子 |
| ------ | ------------ |
|字符串 | `image`, `i` |

## 专辑路径

确定专辑路径。

|类型 |例子 |
| ------ | ------------ |
|字符串 | `album`, `a` |

＃＃ 主机名

* 在使用子域通配符时适用。

确定主机名。

|类型 |示例 |
| ------ | ------------- |
|字符串 | chevereto.com |

## 语言子域

切换语言子域的使用。启用后，将使用像 `es.chevereto.com` 这样的子域而不是 `chevereto.com?lang=es`。

|类型 |效果|
| -------- | ----------------------------- |
|启用|启用语言子域 |
|禁用|禁用语言子域 |

## 用户名子域

切换用户子域的使用。启用后，将使用像 `rodolfo.chevereto.com` 这样的子域而不是 `chevereto.com/rodolfo`。

|类型 |效果|
| -------- | ------------------------ |
|启用|启用用户子域 |
|禁用|禁用用户子域 |
