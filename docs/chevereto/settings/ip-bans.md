# IP屏蔽

`/dashboard/settings/ip-bans`

IP 禁止允许禁止 IP 地址和 IP 范围使用系统。

## 添加 IP 屏蔽

点击“添加IP屏蔽”。

## IP 禁止值

### IP地址

要禁止的 IP 地址。

|类型 |接受 |
| ------ | ----------------------- |
|字符串 | IPV4 和 IPV6 地址 |

### 截止日期

确定 IP 屏蔽的到期日期。

|类型 |可选 |
| -------- | -------- |
|日期时间 |是 |

### 描述

关于类别的简要说明。它将在类别页面的 `<meta name="description">` 标签中使用。

|类型 |接受 |
| ------ | -------- |
|字符串 |任何 |
